# QuickRec PRD

## 1. 개요
- **프로젝트명**: QuickRec (Sophia Hub 내장 도구)
- **작성일**: 2026-03-03
- **목표**: 원클릭으로 화면 녹화 시작/정지. 불필요한 설정 없이 바로 사용
- **타겟 사용자**: sophia (개인용)

## 2. 배경 & 문제 정의

### As-Is (현재 상황)
- Windows 내장 녹화 도구(Game Bar, Win+G)는 게임 녹화 중심으로 설계됨
- 녹화 시작까지 여러 단계 필요 (Game Bar 열기 → 녹화 버튼 찾기 → 클릭)
- 특정 창에서만 동작, 데스크톱 전체 녹화 불가
- QA 업무 시 빠른 화면 녹화가 자주 필요하지만 매번 번거로움

### Pain Point
- 버그 재현 영상을 찍으려면 준비 과정이 너무 많음
- 원하는 순간에 즉시 녹화를 시작할 수 없음
- 별도 녹화 앱(OBS 등)은 오버스펙이고 무거움

### To-Be (기대 효과)
- Sophia Hub에 항상 떠있으므로 즉시 녹화 가능
- 클릭 한 번 또는 단축키 하나로 녹화 시작/정지
- MP4 파일이 자동 저장되어 바로 공유 가능

## 3. 핵심 기능 (MVP)

| # | 기능 | 설명 | 우선순위 |
|---|------|------|---------|
| 1 | 원클릭 녹화 시작/정지 | 버튼 하나로 녹화 토글 | P0 (필수) |
| 2 | 소스 선택 (모니터/창) | 멀티 모니터 환경에서 녹화 대상 선택 (전체 화면 / 특정 창) | P0 (필수) |
| 3 | WebM 자동 저장 | 녹화 종료 시 타임스탬프 기반 파일명으로 자동 저장 | P0 (필수) |
| 4 | 저장 경로 표시 + 열기 | 녹화 완료 후 파일 경로 표시, 클릭하면 폴더 열기 (이미지 저장과 동일 패턴) | P0 (필수) |
| 5 | 글로벌 단축키 | Ctrl+Shift+R로 어디서든 녹화 토글 | P1 (중요) |
| 6 | 녹화 시간 표시 | 녹화 중 경과 시간 실시간 표시 | P1 (중요) |
| 7 | 녹화 파일 목록 | 최근 녹화 파일 목록 표시 + 클릭으로 재생 | P1 (중요) |

## 4. 기술 스택
- **Frontend**: React (Sophia Hub 내장, ToolsTab 컴포넌트)
- **Backend**: Electron Main Process (desktopCapturer, IPC)
- **녹화 엔진**: Electron desktopCapturer + MediaRecorder API
- **파일 포맷**: WebM (MediaRecorder 기본) → 추후 MP4 변환 고려
- **저장 위치**: `~/.claude/recordings/`

## 6. 데이터 모델

### 녹화 파일
- 저장 경로: `~/.claude/recordings/`
- 파일명 패턴: `rec_YYYY-MM-DD_HH-mm-ss.webm`
- 메타데이터: 파일 시스템 기반 (별도 DB 불필요)

### Store 데이터 (electron-store)
```typescript
{
  quickrec: {
    savePath: string       // 저장 경로 (기본: ~/.claude/recordings/)
    hotkey: string         // 단축키 (기본: Ctrl+Shift+R)
  }
}
```

## 7. 사용자 흐름 (User Flow)

### 녹화 시작
1. Sophia Hub의 Tools 탭 열기
2. QuickRec 섹션 펼치기
3. 녹화 소스 선택 (Screen 1 / Screen 2 / 특정 창)
4. "녹화 시작" 버튼 클릭 (또는 Ctrl+Shift+R)
5. desktopCapturer로 선택한 소스 스트림 획득
6. MediaRecorder가 녹화 시작
7. UI에 경과 시간 + 녹화 대상 표시

### 녹화 종료
1. "녹화 중지" 버튼 클릭 (또는 Ctrl+Shift+R)
2. MediaRecorder 종료 → Blob 생성
3. Main process에서 파일로 저장
4. UI에 저장 완료 + 파일 경로 표시 (클릭→폴더 열기)
5. 파일 목록 갱신

## 8. 비기능 요구사항
- **성능**: 녹화 중 Sophia Hub CPU 사용률 10% 이하 유지
- **용량**: 1080p 기준 분당 ~5MB (WebM VP8)
- **안정성**: 비정상 종료 시에도 녹화 파일 손상 방지 (가능한 범위 내)
- **호환성**: Windows 11 (MINGW64 환경)
